# Ordrebog Hertz & Vaaben

En enkel lokal ordrebog til at holde styr på:

- kommende ordrer
- ordrer der er i gang
- færdige ordrer

Åbn `index.html` direkte i en browser, eller kør:

```bash
python3 -m http.server 8000 --bind 127.0.0.1
```

og gå til `http://127.0.0.1:8000/`.

Ordrer gemmes lokalt i browseren.
